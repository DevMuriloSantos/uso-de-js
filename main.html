Módulo A e B

Uso da propriedade window

O objeto window é o objeto global no navegador para JavaScript e oferece várias funções úteis, entre elas:

window.alert()

Uso: Exibe uma mensagem simples ao usuário em uma caixa de alerta.

JavaScript

window.alert("Olá, mundo!");

Características:

  • Exibe apenas uma mensagem e um botão "OK".
  • Não retorna nada (ou seja, retorna undefined).
  • É usada para informar o usuário, mas não para interações.

window.confirm()

Uso: Exibe uma caixa de diálogo com uma mensagem e botões "OK" e "Cancelar".

JavaScript

let resposta = window.confirm("Você tem certeza?");

Características:

  • Retorna um valor booleano:
  • true se o usuário clicar em "OK"
  • false se clicar em "Cancelar"
  • Útil para confirmações de ações (como deletar algo).

window.prompt()

Uso: Exibe uma caixa de diálogo com uma mensagem e um campo de entrada de texto.

JavaScript

let nome = window.prompt("Qual é o seu nome?");

Características:

  • Retorna uma string com o valor inserido, ou null se o usuário clicar em "Cancelar".
  • Útil para coletar informações rápidas do usuário.

Variáveis

No JavaScript, variáveis são usadas para armazenar dados. Você pode declarar variáveis usando três palavras-chave principais:

var – (Forma antiga, evite se possível)

  • Escopo de função
  • Pode ser redeclarada e atualizada
  • Pode causar comportamentos inesperados

JavaScript

var nome = João;
var nome = "Maria";

let – (Recomendado para variáveis que mudam)

  • Escopo de bloco
  • Pode ser atualizada, mas não redeclarada no mesmo escopo
  • Comportamento mais previsível

JavaScript

let idade = 25;
idade = 30; // ok
// let idade = 30; // erro se no mesmo escopo

const – (Recomendado para valores constantes)

  • Escopo de bloco
  • Não pode ser atualizada nem redeclarada
  • Ideal para variáveis que não mudam (como objetos ou funções fixas)

JavaScript

const PI = 3.14;
// PI = 3.1415; // erro

Objetos e arrays criados com const podem ter seus conteúdos modificados, mas não podem ser reatribuídos:

JavaScript

const pessoa = {nome: "Ana" };
pessoa.nome = "Clara"; // ok
// pessoa = { nome: "Lucas" }; // erro

Convenções de nomes de variáveis

  • Use nomes descritivos e em camelCase
  • Evite usar palavras reservadas (como if, while, class).
  • Variáveis não podem começar com números nem ter espaços.

JavaScript

let precoProduto = 100;
const numeroMaximoDeTentativas = 3;

Tratamento de dados

O tratamento de dados com JavaScript envolve diversas operações para limpar, transformar, validar e formatar dados antes de usá-los ou exibi-los. Abaixo estão as técnicas mais comuns organizadas por tipo de dado:

Strings

JavaScript

let nome = " Murilo ";

nome.trim(); // Remove espaços no início e no fim
nome.length; // Verifica a quantidade de caracteres da string, que nesse caso é 6
nome.toUpperCase(); // Converte para maiúsculas -- " MURILO "
nome.toLowerCase(); // Converte para minúsculas -- " murilo "
nome.replace(/MURILO/g, "Murilo"); // Substitui "MURILO" por "Murilo"
nome.includes(" "); // Divide a string em um array -- ["", "Murilo", ""]

Para converter uma string para number em JavaScript, existem várias formas — cada uma adequada para contextos diferentes. Aqui estão as mais usadas:

Usar number()

Converte para número inteiro ou decimal (float).

JavaScript

Number(("123")); // 123
Number("12.34"); // 12.34
Number("abc"); // NaN (Não é um número)

Usar parseInt()

Converte para inteiro, ignorando os decimais.

JavaScript

parseInt("123"); // 123
parseInt("12.34"); // 12
parseInt("123abc"); // 123 (lê até onde consegue)

Usar parseFloat()

Converte para número decimal (float), mantendo os decimais.

JavaScript

parseFloat("12"); // 12
parseFloat("12.34"); // 12.34
parseFloat("abc"); // NaN (Não é um número)

Exemplo de conversão de string para number

JavaScript

let n1 = "10"
let n2 = "15"
teste.innerHTML = 'A soma dos dois números é: ' ${parseInt(n1) + parseInt(n2)}
// Resultado: 25. Considera-se que "teste" refere-se a um id fictício de um parágrafo

JavaScript

let n3 = window.prompt("Digite um número")
let n4 = window.prompt("Digite outro número")
t.innerHTML = 'A soma dos dois números é: ' ${parseInt(n3) + parseInt(n4)}
// Resultado: 25. Considerando que o usuário digitou 10 e 15 respectivamente.

Podemos também fazer a conversão de forma mais direta usando apenas o elemento Number assim já foi citado mais acima:

JavaScript

let n5 = Number(window.prompt("Digite um número"))
let n6 = Number(window.prompt("Digite outro número"))
t.innerHTML = 'A soma dos dois números é: ' ${n5 + n6}
// Resultado: 25. Considerando que o usuário digitou 10 e 15 respectivamente.

Conversão de Number para String

Em JavaScript, existem duas formas mais comuns de converter um Number para String:

Método toString()

JavaScript

let num = 42;
let text = num.toString();
console.log(texto); // "42"

Método String()

JavaScript

let num = 42;
let text = String(num);
console.log(texto); // "42"

Template String

As template strings (introduzida no ES6) são uma forma moderna de trabalhar com strings de maneira mais fácil e intuitiva. Elas permitem a interpolação de variáveis e expressões dentro de strings, utilizando a sintaxe de crase (` `) ao invés de aspas simples ou duplas. Dentro de uma template string, você insere valores com a sintaxe ${variavel}.

Exemplo simples

JavaScript

let nome = "Murilo";
let idade = 15;

let mensagem = `Olá, meu nome é ${nome} e eu tenho ${idade} anos.`;
console.log(mensagem);
// "Olá, meu nome é Murilo e eu tenho 15 anos."

Propriedade toFixed

O método toFixed(n) formata um número para ter exatamente n casas decimais, retornando uma string com o número formatado.

Exemplo

JavaScript

let num = 3.14159

console.log(num.toFixed(2)); // "3.14"
console.log(num.toFixed(4)); // "3.1416"

Perceba:

  • toFixed(2) arredonda para duas casas decimais.
  • o resultado é sempre string, se precisar usar novamente como número, teria que converter de volta.

Outro detalhe importante é que o toFixed() faz arredondameto:

JavaScript

let x = 1.005;
console.log(x.toFixed(2)); // "1.01" (arredondou)

Propriedade toLocaleString

Ela converte números ou datas em strings formatadas conforme a localidade (idioma e país), seguindo padrões culturais (por exemplo, separador decimal, símbolo de moeda, formato de data).

Exemplo com número

JavaScript

let num = 1234567.89;
console.log(num.toLocaleString("en-US")); // "1,234,567.89" (padrão EUA)

console.log(num.toLocaleString("pt-BR")); // "1.234.567,89" (padrão Brasil)

Exemplo formatado como moeda

JavaScript

let preço = 1999.99;
console.log(preço.toLocaleString("pt-BR", {style: "currency", currency: "BRL"})); // "R$ 1.999,99" (padrão Brasil)

console.log(preço.toLocaleString("en-US", {style: "currency", currency: "USD"})); // "$ 1,999.99" (padrão EUA)
