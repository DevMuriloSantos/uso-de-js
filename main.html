Uso da propriedade window

O objeto window é o objeto global no navegador para JavaScript e oferece várias funções úteis, entre elas:

window.alert()

Uso: Exibe uma mensagem simples ao usuário em uma caixa de alerta.

JavaScript

window.alert("Olá, mundo!");

Características:

  • Exibe apenas uma mensagem e um botão "OK".
  • Não retorna nada (ou seja, retorna undefined).
  • É usada para informar o usuário, mas não para interações.

window.confirm()

Uso: Exibe uma caixa de diálogo com uma mensagem e botões "OK" e "Cancelar".

JavaScript

let resposta = window.confirm("Você tem certeza?");

Características:

  • Retorna um valor booleano:
  • true se o usuário clicar em "OK"
  • false se clicar em "Cancelar"
  • Útil para confirmações de ações (como deletar algo).

window.prompt()

Uso: Exibe uma caixa de diálogo com uma mensagem e um campo de entrada de texto.

JavaScript

let nome = window.prompt("Qual é o seu nome?");

Características:

  • Retorna uma string com o valor inserido, ou null se o usuário clicar em "Cancelar".
  • Útil para coletar informações rápidas do usuário.

// --> uma única linha de comentário;

/**/ --> mais de uma linha de comentário.

Variáveis

No JavaScript, variáveis são usadas para armazenar dados. Você pode declarar variáveis usando três palavras-chave principais:

var – (Forma antiga, evite se possível)

  • Escopo de função
  • Pode ser redeclarada e atualizada
  • Pode causar comportamentos inesperados

JavaScript

var nome = João;
var nome = "Maria";

let – (Recomendado para variáveis que mudam)

  • Escopo de bloco
  • Pode ser atualizada, mas não redeclarada no mesmo escopo
  • Comportamento mais previsível

JavaScript

let idade = 25;
idade = 30; // ok
// let idade = 30; // erro se no mesmo escopo

const – (Recomendado para valores constantes)

  • Escopo de bloco
  • Não pode ser atualizada nem redeclarada
  • Ideal para variáveis que não mudam (como objetos ou funções fixas)

JavaScript

const PI = 3.14;
// PI = 3.1415; // erro

Objetos e arrays criados com const podem ter seus conteúdos modificados, mas não podem ser reatribuídos:

JavaScript

const pessoa = {nome: "Ana" };
pessoa.nome = "Clara"; // ok
// pessoa = { nome: "Lucas" }; // erro

Convenções de nomes de variáveis

  • Use nomes descritivos e em camelCase
  • Evite usar palavras reservadas (como if, while, class).
  • Variáveis não podem começar com números nem ter espaços.

JavaScript

let precoProduto = 100;
const numeroMaximoDeTentativas = 3;
