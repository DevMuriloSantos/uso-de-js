Uso da propriedade window

O objeto window é o objeto global no navegador para JavaScript e oferece várias funções úteis, entre elas:

window.alert()

Uso: Exibe uma mensagem simples ao usuário em uma caixa de alerta.

JavaScript

window.alert("Olá, mundo!");

Características:

  • Exibe apenas uma mensagem e um botão "OK".
  • Não retorna nada (ou seja, retorna undefined).
  • É usada para informar o usuário, mas não para interações.

window.confirm()

Uso: Exibe uma caixa de diálogo com uma mensagem e botões "OK" e "Cancelar".

JavaScript

let resposta = window.confirm("Você tem certeza?");

Características:

  • Retorna um valor booleano:
  • true se o usuário clicar em "OK"
  • false se clicar em "Cancelar"
  • Útil para confirmações de ações (como deletar algo).

window.prompt()

Uso: Exibe uma caixa de diálogo com uma mensagem e um campo de entrada de texto.

JavaScript

let nome = window.prompt("Qual é o seu nome?");

Características:

  • Retorna uma string com o valor inserido, ou null se o usuário clicar em "Cancelar".
  • Útil para coletar informações rápidas do usuário.
