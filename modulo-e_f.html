Estruturas de repetição no JavaScript

O que são estruturas de repetição no JavaScript ?

São formas de executar um bloco de código várias vezes, enquanto uma condição for verdadeira ou enquanto houver elementos para percorrer. Isso evita escrever o mesmo código repetidamente.

Pricipais estruturas de repetição no JS

1.for

Usado quando você sabe exatamente quantas vezes quer repetir.

2.while

Usado quando você não sabe quantas vezes vai repetir, apenas enquanto a condição for verdadeira.

3.do...while

Parecido com o while, mas a diferença é que ele executa pelo menos uma vez, mesmo que a condição seja falsa logo no início.

4.for...of

Usado para percorrer valores de arrays ou objetos iteráveis.

5.for...in

Usado para percorrer as chaves (índices ou propriedades) de um objeto ou array.

Estrutura Quando usar? Exemplo prático
for Quando sabe quantas repetições Contar de 1 a 10
while Quando depende de uma condição Esperar um botão ser clicado
do...while Quando precisa executar pelo menos uma vez Menu interativo
for...of Percorrer valores em array Lista de nomes
for...in Percorrer propriedades de objetos Acessar atributos de um objeto

Dica

Sempre tenha cuidado com loops infinitos! Exemplo de um erro clássico:

let i = 0; while (i < 5) {
console.log(i);

Isso nunca para de rodar travando o seu código.

Diferença entre estrutura de condição e de repetição

Na estrutura de condição, você testa uma condição e executa um bloco de código apenas uma vez, se essa condição for verdadeira. Já a estrutura de repetição, serve para repetir um bloco de código várias vezes, enquanto uma condição for verdadeira ou enquanto tiver elementos para percorrer.

Diferença entre a estrutura while e do while

a estrutura while, verifica a condição antes de executar o bloco, ou seja, se a condição for falsa logo no início, o bloco nunca será executado. Já a estrutura do while, executa o bloco pelo menos uma vez, depois verifica a condição, ou seja, mesmo que a condição seja falsa, o programa roda o bloco pelo menos uma vez antes de checar.
