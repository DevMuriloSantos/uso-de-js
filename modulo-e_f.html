Estruturas de repetição no JavaScript

O que são estruturas de repetição no JavaScript ?

São formas de executar um bloco de código várias vezes, enquanto uma condição for verdadeira ou enquanto houver elementos para percorrer. Isso evita escrever o mesmo código repetidamente.

Pricipais estruturas de repetição no JS

1.for

Usado quando você sabe exatamente quantas vezes quer repetir.

2.while

Usado quando você não sabe quantas vezes vai repetir, apenas enquanto a condição for verdadeira.

3.do...while

Parecido com o while, mas a diferença é que ele executa pelo menos uma vez, mesmo que a condição seja falsa logo no início.

4.for...of

Usado para percorrer valores de arrays ou objetos iteráveis.

5.for...in

Usado para percorrer as chaves (índices ou propriedades) de um objeto ou array.

Estrutura Quando usar? Exemplo prático
for Quando sabe quantas repetições Contar de 1 a 10
while Quando depende de uma condição Esperar um botão ser clicado
do...while Quando precisa executar pelo menos uma vez Menu interativo
for...of... Percorrer valores em array Lista de nomes
for...in... Percorrer propriedades de objetos Acessar atributos de um objeto

Dica

Sempre tenha cuidado com loops infinitos! Exemplo de um erro clássico:

JavaScript

let i = 0; while (i < 5) {
console.log(i);
// Esquecer de fazer i++
}

Isso nunca para de rodar travando o seu código.

Diferença entre estrutura de condição e de repetição

Na estrutura de condição, você testa uma condição e executa um bloco de código apenas uma vez, se essa condição for verdadeira. Já a estrutura de repetição, serve para repetir um bloco de código várias vezes, enquanto uma condição for verdadeira ou enquanto tiver elementos para percorrer.

Diferença entre a estrutura while e do while

a estrutura while, verifica a condição antes de executar o bloco, ou seja, se a condição for falsa logo no início, o bloco nunca será executado. Já a estrutura do while, executa o bloco pelo menos uma vez, depois verifica a condição, ou seja, mesmo que a condição seja falsa, o programa roda o bloco pelo menos uma vez antes de checar.

Variáveis simples e compostas

Variáveis simples

São aquelas que guardam apenas um valor por vez. Exemplos de valores simples: números, textos, booleanos (verdadeiro/falso).

JavaScript

let idade = 25; // número
let nome = "Ana"; // texto (string)
let aprovado = true; // booleano

Cada variável guada apenas um valor.

Variáveis compostas

São aquelas que podem guardar vários valores ao mesmo tempo. No JavaScript, os mais comuns são arrays e objetos.

Array (lista ordenada de valores)

Guarda vários valores em uma única variável, com índices (posições) começando em 0.

JavaScript

let numeros = [10, 20, 30, 40];
console.log(numeros[0]); // 10
console.log(numeros[2]); // 30

Aqui numeros é uma variável composta que guarda 4 valores.

Objeto (coleção de pares chave-valor)

Guarda valores com um nome (propriedade) para cada um.

JavaScript

let pessoa = {
nome: "Murilo",
idade: 30,
cidade: "São paulo"
};
console.log(pessoa.nome); // Murilo console.log(pessoa["idade"]); // 30

Aqui pessoa é uma variável composta com várias informações organizadas.

Objetos dentro de objetos

JavaScript

const usuario = {
nome: "Carlos",
endereco: {
rua: "Rua A",
numero: 100
}
};
console.log(usuario.endereco.rua); // Rua A

Verificando propriedades de objetos

JavaScript

"nome" in pessoa; // true
pessoa.hasOwnProperty("idade"); // true

Adicionando mais valores a um array

No JavaScript, existem várias formas de adicionar mais valores a um array já existente.

1. Usando push()

Adiciona valores no final do array.

JavaScript

let numeros = [1, 2, 3];
numeros.push(4); // adiciona o 4 console.log(numeros); // [1, 2, 3, 4]

2. Usando unshift()

Adiciona valores no início do array.

JavaScript

let frutas = ["maçã", "banana"];
frutas.unshift("uva"); // adiciona no começo
console.log(frutas); // ["uva", "maçã", "banana"]

3. Atribuindo direto por índice

Se você souber o índice, pode atribuir diretamente:

JavaScript

let cores = ["vermelho", "azul"];
cores[2] = "verde"; // adiciona na posição 2
console.log(cores); // ["vermelho", "azul", "verde"]

4. Usando splice()

Permite inserir em uma posição específica do array:

JavaScript

let animais = ["cachorro", "gato"];
animais.splice(1, 0, "coelho");
// (posição 1, 0 itens removidos, adiciona "coelho")
console.log(animais); // ["cachorro", "coelho", "gato"]

Como saber o tamanho/quantidade de elementos do array?

No JavaScript, para saber o tamanho (quantidade de elementos) de um vetor (array), você usa a propriedade .length.

JavaScript

let numeros = [10, 20, 30, 40];
console.log(numeros.length); // 4

Método indexOf()

É um método de arrays (e também de strings) que retorna o índice (posição) da primeira ocorrência de um determinado elemento.

Sintaxe

JavaScript

array.indexOf(valorProcurado, índiceInicialOpcional);

  • valorProcurado → O elemento que você quer buscar.
  • índiceInicialOpcional → De onde começar a busca (padrão é 0).

Exemplo com arrays

JavaScript

let frutas = ["maçã", "banana", "uva", "banana"];
console.log(frutas.indexOf("banana")); // 1 (primeira ocorrência/ primerio índice)
console.log(frutas.indexOf("laranja")); // -1 (não existe no array)

O que são funções?

São ações executadas assim que são chamadas ou em decorrência de algum evento.

Uma função pode receber parâmetros e retornar um resultado.

O que são parâmetros?

Parâmetros são variáveis que você define na declaração da função, para receber valores na hora que a função é chamada.

Eles são como “caixinhas” que recebem valores (chamados argumentos) enviados quando você executa a função.

Exemplo

JavaScript

function somar(a, b) {
return a + b;
}
let resultado = somar(5, 3); // a = 5, b = 3
console.log(resultado); // 8

Número de parâmetros diferente do número de argumentos

No JavaScript, se você chamar a função com menos argumentos, os parâmetros restantes ficam como undefined:

JavaScript

function teste(a, b) {
console.log(a, b);
}
teste(10);
// a = 10, b = undefined

Se você chamar com mais argumentos do que parâmetros, os extras são ignorados (mas você pode acessá-los via arguments ou operador rest):

JavaScript

teste(1, 2, 3, 4);
// a = 1, b = 2 (3 e 4 são ignorados diretamente)
