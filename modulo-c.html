O que é DOM?

DOM significa Document Object Model (Modelo de Objeto do Documento). Ele é a representação em árvore de toda a estrutura HTML da página, onde cada tag vira um objeto que você pode acessar e manipular com JavaScript.

Esse exemplo usa o DOM para pegar o elemento h1 pelo id e mudar o texto quando o botão é clicado.

Árvore DOM

A árvore DOM (Document Object Model) representa a estrutura de uma página HTML como uma árvore hierárquica de objetos, onde cada elemento HTML é um nó (node). Essa estrutura permite que o JavaScript acesse e manipule os elementos da página.

Métodos comuns do DOM

Método Descrição
getElementById Retorna o elemento que possui o ID especificado.
getElementsByClassName Retorna uma coleção de elementos que possuem a classe especificada.
getElementsByTagName Retorna uma coleção de elementos que possuem a tag especificada.
querySelector Retorna o primeiro elemento que corresponde ao seletor CSS especificado.
document.querySelectorAll() Seleciona todos que batem com o seletor
document.createElement() Cria um novo elemento
element.innerText Lê ou altera o texto de um elemento
element.innerHTML Lê ou altera o HTML interno
element.style Altera estilos CSS diretamente via JS
element.classList Manipula as classes CSS de um elemento

É importante lembrar que a forma mais nova e recomendável de se utilizar o DOM é através de seletores modernos, como o querySelector e querySelectorAll.

Eventos DOM

No JavaScript, eventos DOM são ações ou ocorrências que acontecem na página da web, como um clique do mouse, o carregamento da página, pressionar uma tecla, entre outros. Com eles, você pode tornar sua página interativa.

Exemplos comuns de eventos DOM

Evento Descrição
click Quando o elemento é clicado
mouseover Quando o mouse passa por cima
mouseout Quando o mouse sai de cima
keydown Quando uma tecla é pressionada
keyup Quando uma tecla é solta
submit Quando um formulário é enviado
change Quando o valor de um input muda
load Quando a página é carregada
scroll Quando a página é rolada
