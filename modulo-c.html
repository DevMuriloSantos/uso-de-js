Módulo C

O que é DOM?

DOM significa Document Object Model (Modelo de Objeto do Documento). Ele é a representação em árvore de toda a estrutura HTML da página, onde cada tag vira um objeto que você pode acessar e manipular com JavaScript.

Esse exemplo usa o DOM para pegar o elemento h1 pelo id e mudar o texto quando o botão é clicado.

Métodos comuns do DOM

Método Descrição
getElementById Retorna o elemento que possui o ID especificado.
getElementsByClassName Retorna uma coleção de elementos que possuem a classe especificada.
getElementsByTagName Retorna uma coleção de elementos que possuem a tag especificada.
querySelector Retorna o primeiro elemento que corresponde ao seletor CSS especificado.
document.querySelectorAll() Seleciona todos que batem com o seletor
document.createElement() Cria um novo elemento
element.innerText Lê ou altera o texto de um elemento
element.innerHTML Lê ou altera o HTML interno
element.style Altera estilos CSS diretamente via JS
element.classList Manipula as classes CSS de um elemento
